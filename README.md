Employee List Application

 Create Employee
 Update/Edit Employee
 View Employee Details
 Remove an Employee
 View All Employees

 Techs Used
 Backend - C# .NET Web API / MSSQL Database
 Frontend - Angular 

 Sa pagru-run po ng project, I assumed po na meron na kayong ng mga need para mastart ang project Angular CLI, .NET SDK, MSSQL etc.

 The Main folder has 2 folders which are the c# and the angular folder.

 To run/start 
   For the backend (C# API folder): punta lang po sa terminal ng mismong C# folder then run the command: dotnet run
   For the frontend (ANGULAR UI folder): Punto po ulit sa terminal ng mismong Angular UI folder then run the following commands: npm install (para marun and project) and then ng serve (actually runnin the project)

   backend: dotnet run
   frontend: npm install, ng serve

 Then open na lang po yung url kung saan magrrun ang Angular which is: http://localhost:4200
 Dito napo matetest yung functionalities nung app:
 Create Employee
 Update/Edit Employee
 View Employee Details
